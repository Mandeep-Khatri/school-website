# Nishchal English Boarding School Website

This is the official website project for **Nishchal English Boarding School**.

## Features

- School profile and mission
- Leadership section
- Programs section (K-8)
- Admissions contact actions (call/email)
- Photo slideshow that changes every 3 seconds
- Responsive layout for desktop and mobile

## Project Files

- `index.html` - Main website page
- `styles.css` - Styling and layout
- `script.js` - Interactions and slideshow logic
- `*.jpg` - School logo and photo assets

## Run Locally

From the project folder:

```bash
python3 -m http.server 8000
```

Open:

`http://localhost:8000`
